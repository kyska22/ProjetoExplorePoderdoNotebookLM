# ProjetoExplorePoderdoNotebookLM
Durante a preparação para minha primeira certificação de Scrum, utilizei o NotebookLM como ferramenta de apoio para organizar e consolidar o conteúdo de estudo.   Com ele, reuni materiais, gerei resumos automáticos, transformei áudios em sínteses e criei flashcards que facilitam a revisão rápida.
