# 📚 Estudo para Certificação Scrum com NotebookLM

## 📌 Visão Geral

Este projeto demonstra como ferramentas de **Inteligência Artificial** podem apoiar o processo de aprendizagem durante a preparação para certificações profissionais.

Durante a preparação para minha primeira certificação em Scrum, utilizei o **NotebookLM** para organizar materiais de estudo, gerar resumos automáticos, processar conteúdos em áudio e criar flashcards para facilitar a revisão dos conceitos.

O objetivo é transformar informações dispersas em **conhecimento estruturado**, tornando o estudo mais eficiente e acessível em qualquer lugar.

## 🧠 Ferramenta Utilizada

* Google NotebookLM

O NotebookLM foi utilizado para:

* Organizar fontes e materiais de estudo
* Gerar resumos a partir de documentos
* Extrair ideias principais de conteúdos em áudio
* Criar flashcards para revisão rápida
* Consolidar conceitos importantes para a certificação

## 🎯 Objetivo do Projeto

Demonstrar uma aplicação prática de IA para:

* Organização do conhecimento
* Geração automática de resumos
* Reforço de aprendizagem
* Otimização do estudo para certificações profissionais
